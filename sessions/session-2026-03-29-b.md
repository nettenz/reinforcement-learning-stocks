### 3) Action diversity bonus (anti-collapse)
- Added `reward_action_bonus_scale` parameter to `TradingEnv`
- Gives small bonus when agent takes Buy/Sell instead of Hold
- Prevents policy from collapsing to "always hold"
- Default value: `0.02`
- Results with action bonus (`0.03`):
  - Collapse rate: 33% → **0%**
  - Mean val accuracy: 0.31 → **0.47**
  - Mean test accuracy: 0.33 → **0.49**
  - Best val accuracy: 0.50 → **0.58**

### 4) P&L chart hover improvements
- Added persistent label on mouse hover
- Shows vertical rule at hovered point
- Displays highlighted circle marker
- Label format: `P&L: $X.XX | Net worth: $X.XX`

### 5) Accuracy improvement workflow documented
- Added workflow loop to README:
  1. Experiment Insights → run recommendations
  2. Experiment Insights → check trends
  3. Signal Analytics → deep-dive promising runs
  4. Experiments → manual tweaks
